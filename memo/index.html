Mathオブジェクト

切り捨て･切り上げ系

Math.round() 四捨五入
Math.ceil() 切り上げ
Math.floor() 切り捨て
Math.trunc() 小数点以下を切り取る

Math.floor()とMath.trunc()の違い

				console.log(Math.floor(-1.234)) //切り捨て = 数値以下の最大の整数を返す
				> -2
				console.log(Math.trunc(-1.234)) //単純に小数点以下をなくす
				> -1

その他のMathオブジェクトのメソッド

Math.abs() 絶対値
Math.sqrt() 平方根(√)
Math.cbrt() 立方根(3乗根、3√)
Math.exp() e(ネイピア数)の累乗
Math.log() 自然対数(loge)
Math.log2() 底を2とした対数(log2)
Math.log10() 底を10とした対数(log10)
Math.max() 入力された値の最大値を返す
Math.min() 入力された値の最小値を返す
Math.pow() べき乗、つまりab
Math.random() 0以上1未満の乱数を返す
Math.sign() 入力された値の符号を返す

プロパティとメソッドの違い

  • プロパティ
    ↳そのオブジェクトの持つ情報、変数
  • メソッド
    ↳オブジェクト内の関数、実行プログラム

例

				let object = {
				 //プロパティ = オブジェクトの情報、変数など
				 property1 = "dataData",
				 property2 = 1234567890n,
				 //メソッド = 実行するプログラム
				 method1 = () => {document.getElementById("input").value = 20},
				 method2 = val => {localStorage.setItem("data",val)}
				};
